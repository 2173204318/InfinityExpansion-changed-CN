# InfinityExpansion 无尽贪婪 - 魔改版

此附属增加了许多的游戏后期物品，比如需要消耗大量资源的"无尽锭"。无尽锭可以用来制作功能强大的装备与机器。

魔改版修改了很多内容，而且对于某些服务器可能存在不兼容的情况，建议安装前先进行备份测试以免出现问题。
